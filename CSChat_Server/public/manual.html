실시간 채팅 애플리케이션 가이드

1. 앱 개요

CSChat은 실시간 채팅 기능을 제공하는 웹/모바일 기반 커뮤니케이션 애플리케이션입니다. 개인 메시지, 그룹 채팅, 전체 공지 등 다양한 소통 방식을 지원하며, 효율적인 팀 협업을 가능하게 합니다.

1.1 주요 특징

⚡ 실시간 메시징

WebSocket 기반의 지체 없는 실시간 대화 지원

👥 그룹 협업

사용자 자유 생성 그룹 채팅 및 1:1 대화 지원

🔔 통합 알림

윈도우 토스트, 상단 배너 등 입체적 알림 제공

🛡️ 강력한 관리

대화 모니터링 및 기기 바인딩을 통한 보안 관리

1.2 시스템 요구사항

구분 항목 요구사항
서버 운영체제 Windows 10/11, Win Server 2016+
RAM 4GB 이상 권장
디스크 10GB 이상 여유 공간
네트워크 고정 IP 권장, 3001 포트 개방
클라이언트 Windows Windows 10 이상
Android Android 5.0 (Lollipop+)
Browser Chrome, Edge, Safari 최신 버전

2. 설치 및 실행

2.1 서버 설치 (관리자)

인스톨러 다운로드

(주)조은시스템으로부터 서버 인스톨러 프로그램 다운로드

기기 인증

인스톨러 실행 후 생성된 UUID를 제공업체에 전달하여 인증키 발급

인증 완료 및 설치

전달받은 인증키 입력 후 설치 진행. 완료 시 서버 자동 실행

접속 확인

브라우저에서 http://localhost:3001 접속 확인

⚠️
라이센스 주의사항 인증키는 기기의 UUID와 연결되어 있으므로, 서버 PC 교체 시 재인증이 반드시 필요합니다.

2.4 접속 주소 안내

🌐 랜딩 페이지

http://[서버IP]:3001

일반 사용자용 다운로드 및 접속

⚙️ 관리자 페이지

http://[서버IP]:3001/admin

시스템 제어 및 모니터링

3. 로그인 및 계정

3.1 계정 체계

CSChat은 보안을 위해 관리자가 중앙에서 계정을 생성하고 배포하는 폐쇄형 운영 방식을 따릅니다.

3.4 기기 바인딩 (Security)

🔒
작동 방식 특정 계정이 등록된 기기에서만 접속 가능하도록 제한합니다. 다른 기기에서 로그인 시도시 즉시 차단되어 계정 탈취를 방지합니다.

4. 주요 기능

  • 전체 대화방: 모든 구성원이 참여하는 공식 소통 창구
  • 그룹 대화방: 프로젝트 또는 팀 단위로 멤버를 선택하여 생성
  • 1:1 대화: 개인 간의 프라이빗한 메시지 교환
  • 읽음 표시: 실시간 상대방 확인 여부 추적

5. 관리자 기능 (Admin Only)

관리자 계정으로 로그인 시 상단 메뉴를 통해 접근 가능합니다.

📊 대시보드

총 사용자, 실시간 접속자, 일일 트래픽 및 DB 가용량 모니터링

👁️ 대화 모니터링

조직 보안을 위해 진행 중인 모든 대화 내용 실시간 관제

📢 긴급공지

전체 사용자 대상 팝업 공지 및 미확인자 대상 재발송 알림

5.3 사용자 관리 주요 항목

  • 사용자 기기 정보(OS, 기기명) 자동 수집
  • 비밀번호 강제 초기화 (기본값 1234)
  • 계정별 기기 바인딩 활성화/비활성화
  • 관리자 권한 부여 및 회수

6. 고급 기능

6.1 스마트 알림 시스템

창 자동 팝업

중요 메시지 수신 시 최소화된 창이 자동으로 활성화됨

상단 배너 알림

대화 중에도 다른 방의 알림을 방해 없이 상단 배너로 확인

7. 문제 해결 FAQ

Q. 로그인 실패 오류가 계속됩니다.
비밀번호가 맞음에도 실패한다면 '기기 바인딩'이 설정되어 있을 수 있습니다. 관리자에게 기기 해제를 요청하세요.
Q. 서버 연결이 되지 않습니다.
서버 PC의 IP가 변경되었는지 확인하거나, 방화벽에서 3001 포트가 차단되어 있는지 네트워크 담당자에게 확인하십시오.

9. 기술 사양

Backend Node.js (Express.js)
Database SQLite3 (Journal Mode: WAL)
Protocol WebSocket (Socket.IO)
Encryption bcryptjs (Salt Rounds: 10)
